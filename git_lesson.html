Lesson 2 - Initialize Git! 

Create a Project folder in your computer.

Add an empty text document to that folder. It can be named anything you want. //Make sure you do this

	If you're using CMD or Powershell make sure you understand the following commands:

Lesson 3 - Add your files to your git repository!




Run the command:
	git status
	
You should see that the file that we have created in our project folder is considered "untracked". 
This means that git is not current keeping track or updates on that file. 
If we want to track the file we would have to run the git add command.

Run the command:
	git add "InsertFilenameHere.txt"
	
	This adds the specific file to the git repository

The other command you can run is:
	git add "*"
	
	This adds all current untracked files to be tracked by git.
	
Run the command:
	git status
	
It should tell you that changes are ready to be committed. 
This means that the file has been added to the repository,
but the current changes to the file has not yet been made.

Run the command 
	git commit -m "Init Commit"
	
The git commit saves or updates the current file to the repository. 
The -m "Init Commit" save the "Init Commit" as a message to go with the update.





Lesson 4 - Create a new github repository




Go to your github account.

Select new Repository and name it whatever you want.

Got back to your terminal and then add the remote repository with:

git remote add origin place-repository-URL-here

In order to verify the repository we run:
git remote -v

Finally we run:
git push origin master

That pushes the repository you have in your computer to github.

Exercise: 
Try making changes to your text file and then push those changes to github!

Lesson 6 - Github Pages




Github has this thing where it lets you host a page with it.

So...... Let's make one.

Head over to GitHub and create a new repository named username.github.io, 
where username is your username on GitHub.
example: lcarrico.github.io

Go ahead and clone the new repository you just made

cd into your new repositroy

Create the index.html file inside the that you will be using.

Here's an example:



<html>
	<head></head>
	<body>
		<h1> Welcome to my site! </h1>
	
		<h3> By Leo Carrico </h3>
		
		<p> I'm just a regular boy living life and doing regular things. </p>
	</body>
</html>





Add and commit this file to the repository and then push it to github with the command:
git push

Now open your web browser and check out the site that you made! 
(note: site should url should look like "https://username.github.io.")


Go ahead and try to improve your site a bit.
