Problem 1

Description

Your program must output true if the given number n is a tall number and print false if n is not a tall number. A tall number is a number that every digit is bigger or equal to the left digit. Example: 123 is a tall number because: 3 >= 2 2 >= 1 Note: The first digit can be 0

Input

Line 1: An integer n .

Output

A single line containing true or false

Example 

 
		Input		Output
		123		true

Problem 3

Description

The existence of Lychrel number is an open mathematical problem. Given a number N, repeat the process of reversing digits and adding the resulting number, until forming a palindrome. If the process ends, the number is not a Lychrel number but for example, 196 is supposed to be a Lychrel number because the process seems not to end.

Input

N

Output

List of numbers "N N1 N2..." separated by space, resulted from this process until forming a palindrome

Example 

 
		Input		Output
		10		10 11

Problem 5

Description

You must print the greatest number using all the characters in the input, including the ten digits 0 to 9, the minus sign - and the decimal dot .. Beware: * The dot alone must not end a number, at least a digit is needed. For example, 98742. is refused, write 9874.2 instead. * Trailing and leading zeros must be removed. Write -4 instead of -4.0000 and -5.65 instead of-5.6500.

Input

Line 1 : The number N of chars Line 2: N chars on a single line separated with spaces

Output

The greatest number possible using all the input chars (but maybe without some zeros).

Example 

 
		Input		Output
		8		-2.45889
		4 9 8 . 8 5 2 -
