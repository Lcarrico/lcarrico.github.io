Problem 1

Description

Call a number num strange if there is such a number X so if you perform the operation X to the power of X plus X, you will get num. In the problem, you must check if a number is strange.

Input

An integer num.

Output

True if num is a strange number and False otherwise.

Example 

 
		Input		Output
		6		True

Problem 2

Description

Multiply all of the digits of N, and repeat the same with the product obtained till the product cosists of only one digit. Output the number of steps taken to do so. Example: N = 39 Step 1: 3 * 9 = 27 Step 2: 2 * 7 = 14 Step 3: 1 * 4 = 4 Since it took us 3 steps to reach a number with only 1 digit, the output is 3.

Input

Line 1 : An integer N

Output

The multiplicative persistence of N, as described in the example above

Example 

 
		Input		Output
		39		3
